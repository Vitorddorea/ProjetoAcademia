# 🏋️ Sistema de Gerenciamento de Academia

## 📌 Sobre o Projeto

O Sistema de Gerenciamento de Academia é um projeto desenvolvido em Java com foco na organização e administração de uma academia. O sistema permite o gerenciamento de alunos, instrutores, aulas e inscrições, facilitando o controle das informações de forma prática e organizada.

Este projeto foi desenvolvido com o objetivo de aplicar conceitos de Programação Orientada a Objetos (POO), modelagem de banco de dados, tratamento de exceções e integração com banco de dados PostgreSQL.

---

# 🚀 Funcionalidades

* Cadastro de alunos
* Cadastro de instrutores
* Cadastro de aulas
* Realização de inscrições em aulas
* Listagem de informações cadastradas
* Atualização de dados
* Exclusão de registros
* Tratamento de exceções
* Integração com banco de dados PostgreSQL
